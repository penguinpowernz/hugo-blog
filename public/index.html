Too many disks?

14 Jun 2016, 13:43

I sometimes get confused with how many disks I have connected and which is which. There is the lsblk command which is cool and prints output in a tree format: NAME MAJ:MIN RM SIZE RO TYPE MOUNTPOINT sda 8:0 0 238.5G 0 disk ├─sda1 8:1 0 243M 0 part /boot ├─sda2 8:2 0 1K 0 part └─sda5 8:5 0 238.2G 0 part ├─ubuntu--vg-root 252:0 0 222.3G 0 lvm / └─ubuntu--vg-swap_1 252:1 0 16G 0 lvm [SWAP] sdb 8:16 0 232.9G 0 disk └─sdb1 8:17 0 232.9G 0 part sdc 8:32 0 465.8G 0 disk sr0 11:0 1 1024M 0 rom However, I can’t see the model names of the disks in there.
